# FAQ - Часто задаваемые вопросы

## Откуда берутся новости?

Новости автоматически загружаются из RSS-ленты Resource World:
https://resource-world.ru/forums/-/index.rss

Парсинг происходит в реальном времени при загрузке страницы.

## Как работают рекомендации?

Система рекомендаций анализирует ваши предпочтения (ключевые слова, которые вы добавляете) и подбирает статьи по следующим критериям:

1. **Совпадение в заголовке** - 2 балла
2. **Совпадение в описании** - 1 балл
3. **Совпадение в категории** - 3 балла

Статьи сортируются по сумме баллов, и вам показываются наиболее релевантные.

## Можно ли читать статьи офлайн?

Нет, для загрузки новостей требуется подключение к интернету. Однако уже загруженные статьи можно просмотреть до перезагрузки страницы.

## Как добавить предпочтения?

1. Перейдите в раздел "Мои предпочтения" на главной странице
2. Введите ключевое слово (например: JavaScript, React, Python)
3. Нажмите кнопку "Добавить" или клавишу Enter
4. Предпочтение сохранится в браузере

## Как удалить предпочтение?

Нажмите на крестик (×) рядом с названием предпочтения в списке.

## Где хранятся мои данные?

Все данные (предпочтения, статус cookie) хранятся локально в вашем браузере через LocalStorage. Мы не передаем данные третьим лицам.

## Почему некоторые статьи не показываются?

Система фильтрует контент и показывает только статьи, исключая:
- Файлы для скачивания
- Ресурсы
- Короткие описания (менее 50 символов)

## Как открыть оригинальную статью?

В режиме просмотра статьи нажмите кнопку "Читать оригинал на Resource World" - она откроется в новой вкладке.

## Какие браузеры поддерживаются?

Приложение работает во всех современных браузерах:
- Chrome (последние версии)
- Firefox (последние версии)
- Safari (последние версии)
- Edge (последние версии)

## Как сообщить об ошибке?

Создайте Issue в репозитории GitHub с подробным описанием проблемы.
