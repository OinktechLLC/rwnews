# Условия использования

Последнее обновление: 2024

## 1. Общие положения

1.1. Сервис RW News IT предоставляет доступ к новостям из открытых RSS-источников, в частности с сайта Resource World (https://resource-world.ru).

1.2. Использование сервиса означает полное согласие с настоящими Условиями использования.

1.3. Администрация оставляет за собой право вносить изменения в Условия использования без предварительного уведомления.

## 2. Права на контент

2.1. Все права на новости, статьи и другие материалы принадлежат их авторам и ресурсу Resource World.

2.2. Сервис RW News IT не создает собственный контент, а только агрегирует информацию из открытых источников.

2.3. Запрещено использование контента в коммерческих целях без разрешения правообладателей.

## 4. Ограничения использования

4.1. Запрещено:
- Использовать сервис для распространения вредоносного ПО
- Осуществлять автоматический парсинг данных сервиса
- Использовать сервис в нарушение законодательства

4.2. Пользователь обязуется не нарушать авторские права владельцев контента.

## 5. Отказ от ответственности

5.1. Сервис предоставляется "как есть" без каких-либо гарантий.

5.2. Администрация не несет ответственности за:
- Точность и актуальность информации в статьях
- Временную недоступность сервиса
- Убытки, возникшие в результате использования сервиса

5.3. Информация в статьях может не соответствовать действительности. Проверяйте информацию в оригинальных источниках.

## 6. Конфиденциальность

6.1. Сбор и обработка персональных данных осуществляется в соответствии с Политикой конфиденциальности.

6.2. Сервис использует cookie для улучшения работы и персонализации.

## 7. Прекращение доступа

7.1. Администрация вправе заблокировать доступ пользователю при нарушении Условий использования.

7.2. Доступ может быть временно ограничен для проведения технических работ.

## 8. Разрешение споров

8.1. Все споры решаются путем переговоров.

8.2. При невозможности достижения согласия спор передается на рассмотрение в суд по месту нахождения истца.

## 9. Заключительные положения

9.1. Настоящие Условия вступают в силу с момента начала использования сервиса.

9.2. Вопросы и предложения направляйте через репозиторий GitHub.
