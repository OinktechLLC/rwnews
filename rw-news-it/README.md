# 📰 RW News IT

Современное веб-приложение для чтения новостей с портала [Resource World](https://resource-world.ru/).

![RW News IT Banner](https://img.shields.io/badge/RW-News%20IT-blue?style=for-the-badge)
![License](https://img.shields.io/badge/license-MIT-green?style=for-the-badge)
![React](https://img.shields.io/badge/React-18-blue?style=for-the-badge&logo=react)

## ✨ Особенности

- 🔄 **Автоматическое обновление** новостей из RSS-ленты Resource World
- 🎯 **Умные рекомендации** на основе ваших предпочтений
- 📱 **Адаптивный дизайн** для всех устройств
- 🔒 **Конфиденциальность** - все данные хранятся локально
- 🍪 **Cookie consent** - управление согласием на файлы cookie
- 📄 **Полная документация** - FAQ, условия использования, политика конфиденциальности

## 🚀 Быстрый старт

### Вариант 1: Простой запуск

1. Откройте файл `public/index.html` в любом современном браузере
2. Наслаждайтесь чтением новостей!

### Вариант 2: Локальный сервер (рекомендуется)

```bash
# Используя Python 3
cd rw-news-it
python3 -m http.server 8000

# Или используя Node.js
npx serve public
```

Затем откройте `http://localhost:8000` в браузере.

## 🛠 Технологии

- **Frontend**: React 18, JSX
- **Стили**: CSS3 с CSS Custom Properties
- **Данные**: RSS парсинг через CORS-прокси
- **Хранение**: LocalStorage для пользовательских данных
- **CORS Proxy**: api.allorigins.win

## 🧠 Умные рекомендации

Система рекомендаций анализирует:

1. **Выбранные темы** - вручную указанные пользователем интересы
2. **Историю просмотров** - какие статьи читал пользователь
3. **Актуальность** - свежесть публикаций
4. **Релевантность** - совпадение ключевых слов

## 🔐 Конфиденциальность

Приложение собирает минимальный объем данных:

- ✅ Предпочтения по темам (хранятся в localStorage)
- ✅ История просмотров (хранится в localStorage)
- ✅ Технические данные (тип браузера)

**Мы НЕ собираем:**
- ❌ Личные данные
- ❌ Email адреса
- ❌ Пароли
- ❌ Данные передаются на сервер

Все данные хранятся **локально** в вашем браузере.

## 🍪 Cookie

Приложение использует файлы cookie для:
- Сохранения пользовательских настроек
- Персонализации контента
- Анализа использования

Вы можете принять или отклонить cookie при первом посещении.

## 🌐 RSS Источник

Приложение использует RSS-ленту:
```
https://resource-world.ru/forums/-/index.rss
```

Фильтруются только статьи (исключаются файлы и ресурсы).
