# RW News IT

Современный RSS-агрегатор новостей с сайта Resource World с умными рекомендациями и персонализацией.

## 🚀 Функции

- ✅ **RSS парсинг** из https://resource-world.ru/forums/-/index.rss
- ✅ **Фильтрация статей** - показываются только статьи, исключаются файлы и ресурсы
- ✅ **Умные рекомендации** - персонализация на основе предпочтений пользователя
- ✅ **Современный дизайн** - адаптивный интерфейс с CSS переменными
- ✅ **Просмотр статей** - чтение внутри приложения + ссылка на оригинал
- ✅ **Cookie баннер** - уведомление о сборе данных с согласием
- ✅ **Документация** - FAQ, Условия использования, Политика конфиденциальности

## 🛠 Технологии

- **React 18** - библиотека для создания интерфейса
- **Vite** - быстрый сборщик проектов
- **CSS Variables** - темизация и адаптивность
- **DOMParser** - парсинг XML/RSS
- **LocalStorage** - хранение предпочтений и настроек

## 📦 Установка

```bash
# Клонирование репозитория
git clone <ваш-репозиторий>
cd rw-news-it

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Предварительный просмотр сборки
npm run preview
```

## 🎯 Использование

### Добавление предпочтений
1. В разделе "Мои предпочтения" введите ключевые слова (например: JavaScript, React, Python)
2. Нажмите "Добавить" или Enter
3. Система автоматически подберет релевантные статьи

### Чтение статей
- Кликните на карточку статьи для просмотра
- Используйте кнопку "Читать оригинал" для перехода на сайт Resource World

### Настройки cookie
- При первом посещении появится баннер с запросом согласия
- Нажмите "Принять" для продолжения работы

## 🔌 API

### RSS Источник
- URL: `https://resource-world.ru/forums/-/index.rss`
- Формат: RSS 2.0

### CORS Proxy
- URL: `https://api.allorigins.win/raw?url=`
- Назначение: обход CORS ограничений браузера

## 🍪 Сбор данных

Приложение собирает минимальные данные:
- Предпочтения пользователя (ключевые слова)
- Статус согласия на cookie

Все данные хранятся локально в браузере (LocalStorage) и не передаются третьим лицам.
